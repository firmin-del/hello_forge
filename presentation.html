Contexte

Pourquoi le Cross-Platform ?

❌ Natif classique

  • 2 équipes séparées
  • Kotlin/Java pour Android
  • Swift/ObjC pour iOS
  • 2x le budget
  • 2x le temps de développement

✅ Flutter

  • 1 seule base de code
  • Android + iOS + Web + Desktop
  • Moteur graphique Impeller/Skia
  • Performances natives
  • Hot Reload instantané
Le Langage Dart

AOT vs JIT

⚡
AOT — Ahead-Of-Time
Compilation avant l'exécution en code machine natif.

→ Production
→ Performances maximales
→ Démarrage instantané
🔄
JIT — Just-In-Time
Compilation pendant l'exécution à la volée.

→ Développement
→ Hot Reload < 1 seconde
→ Itération rapide
